## 🚗 Used Car Price Predictor

### 🔍 Description
Predicting a car's **selling price** based on its features.

### 🧠 Features
- 🚘 Dynamic Input Fields
- 📊 Preprocessing with Label Encoding & Scaling
- 💰 Real-time Price Prediction
- Clean, responsive UI

### To Run:
    ```bash
    streamlit run app.py
    ```
